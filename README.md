# Flourishing City — Assumptions Ledger

**This is the product** (§k of the scope): a numbered, sourced, and *evidence-graded*
(GRADE) set of every parameter in the Flourishing City agent-psychology model — a
mutually consistent set of beliefs about what humans need and what embedded AI does
to those needs. It has standing value independent of whether the simulation runs
convincingly, because it is the thing you can argue with.

- **`ASSUMPTIONS.md`** — the rendered ledger (human-readable).
- **`index.html`** — the same ledger as a page (published via GitHub Pages).
- **`parameters.yaml`** — the machine-readable source of truth. Both views render
  from this file, so they cannot drift.

Coverage: **89/89** scientific scalars. Honest headline: **~83% are graded
`assumed`** — invented modelling scalars with no direct empirical basis. That is the
real state of the model, shown rather than hidden. Sensitivity ranks (which of those
assumed parameters actually matter) are populated as the sensitivity analysis runs.

The simulation engine itself lives in a separate repository. This is a
structured-argument instrument, not a forecast: it generates evidence about a model,
never about the world.
