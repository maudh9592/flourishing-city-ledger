Flourishing City · the ledger is the product (§k)

A numbered, gradeable set of beliefs about what humans need — and what embedded AI does to those needs.

A mutually consistent, sourced, evidence-graded set of assumptions for a synthetic population. It has standing value even if the simulation never runs convincingly — it is the thing you can argue with. Every scalar in the engine maps to one row.

Declared value judgment — read first

The capability list used for the objective wellbeing index (§c.6) is a NORMATIVE COMMITMENT, not a scientific finding. Nussbaum's list is philosophically derived; Sen declined to fix one. This model fixes a list for tractability. That choice is contestable; the list is swappable via config so a critic can substitute their own and re-run. This judgment appears at the TOP of the ledger by design, not buried.

How to argue with this

Every grade is contestable and every value is swappable via config. Disagree with the capability list? Substitute your own and re-run. Think a grade is too generous? The rationale is stated. Think a number is load-bearing and unmeasured? It is probably already on the research agenda — and if not, that is the finding. A structured-argument instrument, not a forecast.
